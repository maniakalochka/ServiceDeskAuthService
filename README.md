# Auth Service for Service Desk

Для настройки сервиса необходимо создать файл `.env` в корневой папке проекта и заполнить его по образцу `.env.example`.
Сгенерировать уникальные значения для секретных ключей можно с помощью следующей команды в терминале:

```sh
python3 -c "import secrets; print(secrets.token_hex(32))"
```
